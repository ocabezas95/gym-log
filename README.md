# gym-log CLI

## Description
A command-line application that allows users to log workouts(exercise, sets, reps, weight, date) and view workout history

## Why I Built This
As someone who trains consistenly, I wanted a lightweight tool to track workouts and eventually integrate AI-based progression insights

## Features (v1)
- Add workout
- view workout
- Data saved locally
- input validation

## Future Improvements
- web interface (FastAPI)
- Database integration
- AI-based weight progression prediction
- Authentication
- Dashboard visualization
